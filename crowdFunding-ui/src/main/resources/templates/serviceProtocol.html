1. 术语和定义

除非本协议正文另有明确约定，在本协议中所用术语定义如下：

  1. 1.1 众筹：指发起人与支持者共同完成项目、实现梦想的行为。在这一过程中支持者出资支持发起人，发起人完成项目并按承诺向支持者发放回报。
  2. 1.2 京东：北京奥都百升网络科技有限公司及其旗下子公司、分公司、分支机构或相关关联公司统称为“京东”，本协议中提及的应由 “京东”行使的权利和履行的义务，由“京东”内部负责相应区域及业务范围的公司分别履行。
  3. 1.3 京东众筹平台：指由京东运营和管理的，提供平台网络空间等技术服务的网络服务平台，包括但不限于域名为z.jd.com的众筹信息服务平台、京东关联方运营/管理的网络服务平台，及未来可能新设或合作的网络平台等。
  4. 1.4 发起人：指在京东众筹平台上发起众筹项目的自然人、法人或非法人组织。
  5. 1.5 支持者：指出资支持众筹项目，与发起人一起实现梦想的自然人、法人或非法人组织。
  6. 1.6 众筹期限：指发起人确定的众筹项目募集款项的时间。
  7. 1.7 募集总金额：指众筹期限届满时所募集的款项总额。
  8. 1.8 众筹募集成功：指项目众筹期限届满且达到预定募集金额。
  9. 1.9 众筹失败：指众筹出现下列情形之一：
    1. （1） 众筹募集失败，即众筹期限届满但未达到预定募集金额；
    2. （2） 发起人由于能力限制、市场风险、法律风险等各种因素，在众筹期限届满前取消项目；
    3. （3） 发起人由于能力限制、市场风险、法律风险等各种因素，在众筹募集成功后明确表示无法发放回报。
  10. 1.10 项目成功结束：指发起人向支持者发放完所有回报产品。
  11. 1.11 网银在线：指网银在线（北京）科技有限公司，为发起人和京东提供资金代收付服务的第三方支付机构。
  12. 1.12 京东钱包账户：指发起人在使用网银在线提供的资金代收付服务时，网银在线提供的唯一识别服务编号，可设置密码，安装数字证书，并具体使用网银在线服务开展交易，查询交易产生的款项收付账务明细。

3. 众筹项目标准及项目回报规范

  1. 3.1 项目应具有创新性质且具有可执行性，项目目标应当明确、具体、可衡量，如制作一个实物、拍一部微电影或完成一件艺术创作等。
  2. 3.2 发起人不得在无实质项目内容的情况下单纯发起募捐，或发起类似“资助奖学金”、“资助我去旅游”等为满足发起人个人需求的众筹项目。
  3. 3.3 项目不得含有以下内容：
    1. （1） 违反国家法律规定的违禁品，如毒品、枪支弹药及管制刀具相关；
    2. （2） 色情、赌博、暴力、恐怖、反动、政治与宗教相关；
    3. （3） 彩票、博彩相关；
    4. （4） 开办公司、网站、店铺等相关；
    5. （5） 其他国家法律规定和京东网站规定的禁限售等违禁品信息。
  4. 3.4 项目回报应当符合以下要求：
    1. （1） 应当明确、具体，并与项目具有关联性。
    2. （2） 不得包含股权、债券、利息、分红等具有投资回报性质的内容，也不得为现金或用于流通的货币。
  5. 3.5 众筹项目及其回报须符合法律法规的相关规定，京东有权根据众筹项目的实际情形对众筹项目和项目回报提出特殊证明要求。

4. 发起人资格及其行为规范

  1. 4.1 发起人应当是京东平台的注册用户，并在发起项目前完成必要的身份认证和资质认证，发起人须提交依据法律规定其经营业务所必须的文件或证明并加盖发起人公章，包括但不限于营业执照、行政许可证、开户行证明、商标注册证、商标使用授权书、质检报告、报关单、检验检疫证书、产品来源地证明、法人、非法人组织法定代表人身份证正反面复印件等。如证明文件发生任何变更或更新时，发起人须立即通知京东，并于3个工作日内，提交更新后的文件。此外，发起人应在其从事经营活动的主页面醒目位置公开营业执照登载的信息或者其营业执照的电子链接标识。京东将根据申请发起的项目类型不同，对发起人需要满足的其他资格进行限定和要求，建立登记档案，并定期核验更新。
  2. 4.2 若发起人为自然人，应提交其姓名、地址、有效身份证明、有效联系方式等真实身份信息。如证明文件发生任何变更或更新时，发起人须立即通知京东，并于3个工作日内，提交更新后的文件。京东有权对发起人真实身份信息进行核验、登记，建立登记档案，并定期核验更新，核发证明个人身份信息真实合法的标记，发起人应将该标记加载在其从事经营活动的主页面醒目位置。
  3. 4.3 发起人应按照京东的要求开立账户，并按要求使用京东账号和收款账户。
  4. 4.4 同一发起人在京东众筹平台仅能使用同一个京东账号发起众筹项目。如同一发起人使用不同京东账号，应向京东出具书面证明，证明不同京东账号归属于同一发起人并经京东同意。如发起人私自更换京东账号，造成的一切损失由发起人承担。
  5. 4.5 同一发起人发起不同众筹项目的，所有众筹项目的收款账户应保持一致，发起人在平台系统预留的银行账户视为接收所有款项的唯一指定账户。如同一发起人使用不同银行账户收款，应向京东出具书面证明，证明不同银行账户归属于同一发起人。如发起人私自变更收款账户，造成的一切损失由发起人承担。
  6. 4.6 不同发起人使用同一个京东账号发起众筹项目的，应向京东出具书面说明并经京东同意。不同的发起人应就彼此发起的所有项目承担连带责任。
  7. 4.7 公益众筹的发起人应当是合法成立的公益组织，或与合法成立的公益组织开展合作，并在完成项目后的合理时间内上传筹款项用于公益用途的相关凭证。
  8. 4.8 京东有权对发起人进行资质审核，该等资质审核仅为形式审核，不代表京东对发起人提交资质文件及各类信息资料真实性、合法性、准确性、及时性的确认，发起人仍应对其提交资料的真实性、合法性、准确性、及时性，以及项目页面陈述等承担相应的法律责任。在发生支持者投诉、行政机关机构调查、诉讼解决等事项时，发起人还应向京东提交与原件核对一致且加盖发起人公章的纸质复印件。
  9. 4.9 发起人应提交真实、准确的项目资质信息及众筹产品信息，不得含有虚假或者引人误解的内容，不得欺骗、误导支持者（项目信息如有任何更新，应及时向京东众筹平台提交更新后的信息），并自主上传、发布项目。发起人保证上述证明文件发生任何变更或更新时立即通知京东，并于十五个工作日内，提交更新后的文件。京东有权根据申请发起的项目类型不同，对发起人需要满足的其他资格和行为规范进行限定和要求。
  10. 4.10 发起人应妥善保管京东众筹平台的账号和密码，任何情况下发起人须对在该账号下发生的所有操作（包括但不限于信息披露、发布信息、上传图片或视频、网上点击同意或提交各类规则协议等）承担法律责任。
  11. 4.11 发起人应自行承担准备或发布、完成项目而发生的费用，自行缴纳因从事本协议项下行为而产生的相应税款。
  12. 4.12 发起人应当在京东众筹平台公示其营业执照信息、行政许可信息、属于不需要办理市场主体登记情形等信息,或者上述信息的链接标识，并授权京东为保护支持者利益使用发起人在京东网站留存的全部资料（包括但不限于发起人留存的认证信息）。
  13. 4.13 项目开始众筹后，发起人除非存在保护支持者利益等特殊理由，否则不得修改项目页面的实质内容（如产品功能、规格等）。
  14. 4.14 众筹期限内发起人因故需取消项目导致众筹失败的，需向京东提交取消申请，京东审核通过后项目取消，所有已筹集到的众筹款项将由系统退回给支持者，募集款项在3个工作日内支付给支持者，具体收款时间以银行处理系统时间为准。
  15. 4.15 发起人应严格按照项目计划执行，并登录京东众筹管理后台完成如下义务：
    1. （1） 每周及时、主动、自行更新项目进展并保证更新质量；
    2. （2） 发生与支持者权益相关的事项应当在24小时内更新项目进展情况，前述事项包括但不限于修改项目内容、延迟发货等；
    3. （3） 提供一个对外客服电话，保证通信畅通，以便及时处理纠纷；
    4. （4） 对话题区进行维护管理，对京东用户的留言应在24小时内回复；
    5. （5） 接入平台即时通讯工具（即“咚咚”），遵守众筹咚咚服务要求和规则；
    6. （6） 支持者申请取消订单的，应在24小时内审核取消申请并向支持者反馈结果；
    7. （7） 在承诺时间内发货并在发货当日导入快递单号；
    8. （8） 在收到管理后台待办事项通知的24小时内处理完毕；
    9. （9） 根据本协议及平台规则需承担的其他义务。
  16. 4.16 发起人应按照承诺发放项目回报应当符合保障人身、财产安全的要求，不得提供法律、行政法规禁止交易的商品或者服务，对其提供的商品承担质量保证责任。（项目衍生成果及/或相关实物、服务，为支持者开具发票等），发放的项目回报应无质量或权利瑕疵，遵守关于无理由退货方面的相关法律法规的规定。如因项目回报发生纠纷，须按法律法规、《京东众筹回报服务规则》及平台公布的其他相关规定，由发起人与支持者进行沟通解决。
  17. 4.17 发起人应严格按照其承诺及《京东众筹回报服务规则》约定的规则履行回报义务。如项目因故可能产生延期或无法按原定项目计划执行的，发起人应及时通过京东众筹平台及其他途径告知支持者并承担相应的责任。
  18. 4.18 发起人应遵守公平竞争、诚实信用原则，不得通过任何手段、以任何方式自行或者委托他人进行如下行为（包括但不限于虚假交易、自买自卖、伪造网络用户真实消费、体验、刷单、虚假发货等其他违反国家法律、法规和平台规则的行为），否则，京东有权解除合同，并要求发起人承担由此给京东造成的全部损失。
  19. 4.19 发起人应严格遵守反洗钱相关的法律法规，京东在发现可疑交易信息时，有权向有关部门报告。
  20. 4.20 若发起人收款账户信息变更，应提前10个工作日通知京东，否则，因发起人变更账户造成货款支付失败、迟延或错误，损失由发起人自行承担。
  21. 4.21 发起人自行负责回报产品的物流配送，因物流配送导致的损失或纠纷由发起人自行承担责任。

    22. 5. 发起人承诺和保证

    1. 5.1 发起人保证订立本协议和在发起人管理系统提出申请是发起人真实意思表示，保证其具有足够资格订立本协议，其代理人（包括所有操作商家管理系统的人员和项目运营所需的全部雇员、职员、管理者）已获得充分授权，并对代理人的行为承担法律责任，同时发起人保证对其雇员、联系人及其他发起人委派的履行本协议的人员的行为承担连带法律责任。

      4. （4） 发起人自愿签署并严格履行本协议；
      5. （5） 京东依据法律法规等要求及经营需要可能设定的其他条件。
    2. 5.3 发起人承诺不得利用平台从事任何违反法律法规或侵犯任何第三方权利的行为，包含但不限于不得自行或允许任何第三方使用发起人的账号通过京东众筹平台从事与下述相关的行为或发布与下述相关的信息：
      1. （1） 侵犯任何第三方的专利、商标、著作权、商业秘密、肖像权或其他合法权利，或违反任何法律或合同的；

      5. （5） 进行任何危害信息网络安全的行为，故意传播恶意程序或病毒以及其他破坏、干扰正常网络信息服务的行为。
    3. 5.4 发起人承诺维护支持者的独家权益，并在众筹项目运营管理中不从事以下行为：

      京东众筹回报服务规则

      根据《京东众筹项目发起人协议》，为进一步规范发起人行为、保护支持者合法权益，制定本回报规则。本规则一经公示即成为《发起人协议》不可缺少的组成部分，发起人勾选《发起人协议》或使用《发起人协议》项下服务的，即表示已阅读并理解本规则的全部内容，同意受本规则的约束。

      1. 回报的错发、漏发

      1.1 虚拟类回报在发货过程中出现错发、漏发或支持者收到的虚拟类回报存在质量问题的，发起人应在七日内免费予以重发或补发。

      1.2 实物类回报在发货过程中因非支持者原因出现的错发、漏发等情况，按照如下方式处理：
      （1） 若出现漏发情形，发起人应当在支持者告知后立即与支持者进行沟通确认，并在确认后2个工作日内进行重发或其他支持者认可的处理方式处理；
      （2） 若出现错发情形，发起人应当在支持者告知后立即与支持者进行沟通确认，并在确认后2个工作日内进行重发或其他支持者认可的处理方式处理。

      2. 回报的退换服务

      2.1 发起人承诺自支持者收到回报的15天内提供无理由退换货服务（支持者收到回报指支持者在系统对订单做确认收货或发起人在众筹管理后台上传订单的有效发货物流单号15天系统默认订单收货），支持者应保证包装、配件完整，不影响二次使用。如支持者选择退货的（支持者上传有效物流单号15天默认发起人收到退回货品），发起人应于收到退回货物之日起3个工作日内为支持者全额退款。 如支持者选择换货的，发起人应于收到更换货物之日起5个工作日内完成换货处理。非质量原因的退换货行为，由支持者自行承担邮费，或双方协商处理。

      2.2 以下回报不适用于无理由退换货：
      （1） 个人定作类；
      （2） 在线下载或者已经拆封的音像制品，计算机软件等数字化类；
      （3） 交付的报纸期刊类；
      （4） 未经授权的维修、误用、碰撞、疏忽、滥用、进液、事故、改动、不正确的安装所造成的回报质量问题，或撕毁、涂改标贴、机器序号、防伪标记；
      （5） 其他根据回报性质及法律规定并在众筹项目页面明确表示不适用退换货的回报。

      3. 服务响应时效

      在履行项目回报义务时，发起人应积极响应支持者、京东提出的各项问题和需求。对于支持者或京东以各种形式提出的问题、需求（包括但不限于纸质函件、电话、邮件、电子数据形式），发起人应于24小时内回复处理意见或对申请完成处理，未在时效内完成处理，京东有权根据协议规定和公告在发起人众筹管理后台的平台规则要求发人承担责任。

      4. 特别约定

      4.1 若发起人同时在京东商城（网址：www.jd.com）开设有店铺，则发起人在京东众筹平台销售的众筹产品的售后规则应与京东商城店铺内产品的售后规则一致。如与众筹平台规则不一致的，以京东商城店铺内产品的售后规则为准。同时，众筹产品订单将同步至发起人在京东商城的店铺。

      4.2 若发起人为京东商城（网址：www.jd.com）自营商品经营者，则发起人在京东众筹平台销售的众筹产品的售后规则应与京东商城自营产品的售后规则一致。如与众筹平台规则不一致的，以京东商城自营产品的售后规则为准。同时，众筹产品订单将同步自营商品经营者。

      附件二：

      营销推广服务协议

      若发起人使用京东营销系统开展优惠券营销活动（以下简称“优惠活动”），发起人应遵守以下优惠券服务使用约定：

      1、 因众筹支持者参与前述优惠活动而产生的营销费用（即营销实际使用的优惠券的总金额）全部由该众筹项目的发起人承担。

      2、 发起人参加活动方式

      若发起人使用京东营销系统开展优惠券营销活动的，可通过邮件形式发送申请至平台指定邮箱，邮件内容需包含该项目的项目编号、优惠券金额、优惠券活动时间、可使用优惠券数量等信息。

      3. 费用结算

      3.1 费用金额

      优惠活动期满后，京东通过指定邮箱将电子版结算明细发送至发起人指定邮箱,发起人应在收到后3个工作日内通过指定邮箱确认。发起人未在约定时限内确认或提出异议的，视为发起人认可该结算明细及金额。发起人应根据京东众筹平台提供的对账单金额支付相应费用。

      京东众筹平台指定邮箱：zcjs@jd.com

      发起人指定邮箱：发起项目的对接人邮箱，如邮箱有变动的，需要提前3个工作日发邮件至京东平台指定邮箱zcjs@jd.com说明。

      3.2 支付方式及时间

      发起人不可撤销的授权同意京东众筹平台可随时直接向网银在线发出代扣指令，按照附件二的3.1条方式确认的金额从发起人在网银在线的商户账号中直接进行划扣用以支付本协议约定的营销费用。

      3.3 退款退货

      若众筹支持者要求退款退货，应将优惠券以相应形式退还给支持者。该部分的营销费用全部由发起人承担。


      附件三：

      平台技术服务费特别约定：

      1. 技术服务费减免约定：

      （1） 为更好的支持众筹创新项目，众筹募集期间内（即自众筹预热期起至项目成功结束），若发起人能够通过京东众筹平台的推广链接从京东以外的渠道拓展有效用户，京东按照1元/有效用户的标准对平台技术服务费进行减免，减免总金额不超过京东收取的平台技术服务费总额。

      有效用户认定标准：不包含虚假数据和重复数据，京东有权根据用户访问深度、停留时长和活跃度等情况进行单方综合判断。

      （2） 众筹项目成功结束后，京东按照前述标准减免其平台及技术服务费。减免的技术服务费以赠送同等价值的广告资源位的形式提供，可通过京东来客平台账户查询。

      3. 发起人应遵守公平竞争、诚实信用原则，不得通过任何手段、以任何方式自行或者委托他人伪造有效用户，否则京东有权要求发起人返还所有减免的技术服务费，并要求发起人承担由此给京东造成的全部损失。
